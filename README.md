# autoenroll-core

Irish Auto-Enrolment SaaS Platform

## Getting Started

This is a pnpm-based monorepo for the Irish Auto-Enrolment platform.

### Prerequisites

- Node.js >= 18.0.0
- pnpm >= 8.0.0

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

The orchestrator app will be available at http://localhost:3000

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```

## Project Structure

```
apps/
  orchestrator/    # Next.js web application
packages/          # Shared packages (coming soon)
```
